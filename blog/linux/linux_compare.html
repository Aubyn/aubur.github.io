Linux 对比 ​

why ​

Linux 发行版众多，有 Arch 系、Debian 系、RHEL 系、SUSE 系等。

选发行版的本质，一是选择包管理器（软件包生态、活跃度），二是选择其理念（大而全，按需渐进式，配置式），三是选择其适配性（最重要！硬件正常驱动、软件正常运行）。

在适用各发行版后，因没有及时记录使用体验，每次对比时，都回忆不全。想深入了解只能重新安装体验，费时费力。本文的目的也就是记录下来，便于以后对比。

how ​

每个发行版会列举优缺点，根据发行版的本质进行列举。

其次会列举发行版的版本。发行版的优缺点会随更新迭代而变化，过于久远的体验无法展现其当前真正的优缺点。

最后会列举安装发行版的机器配置。不同机器上，驱动支持不全等问题可能导致发行版使用异常。

⏳what ​

○ EndeavourOS | Arch ​

  • 版本：EndeavourOS_Endeavour_neo-2024.09.22.iso
  • 机器：Intel NUC11PAHi5

优点：

  • 衍生自 Arch，生态好，GUI 简化安装
  • 配合 gnome 桌面使用，占用低

缺点：

  • 安装时，LiveCD 较卡顿
  • 睡眠后重新解锁，掉网络

○ Debian ​

  • 版本：debian-12.7.0-amd64-DVD-1.iso
  • 机器：Intel NUC11PAHi5

优点：

  • 稳定，生产可用
  • 资料丰富
  • 软件包丰富。软件一般会提供 Debian 版本（.deb）。可配合 铜豌豆软件源，安装国产软件

缺点：

  • 镜像源更新较慢。当前 Debian 镜像源已经比过往更新频率快挺多

○ Linux Mint | Ubuntu ​

  • 版本：linuxmint-22-cinnamon-64bit.iso
  • 机器：Intel NUC11PAHi5

优点：

  • 衍生自 Ubuntu

缺点：

  • 默认仍使用 X11 而不是 Wayland，Wayland 仍处于实验性阶段

○ ElementaryOS | Ubuntu ​

  • 版本：elementaryos-8.0-stable.20241122rc.iso
  • 机器：Intel NUC11PAHi5

优点：

  • 界面精美，类似 macOS，并制作了一系列同风格的默认软件
  • Pantheon DE 基于 GNOME
  • 注重隐私和安全，默认不包含遥测，提供不错的权限管理
  • 衍生自 Ubuntu，软件包丰富

缺点：

  • 睡眠后重新解锁，掉蓝牙，需要手动重新连接

○ KDE neon | Ubuntu ​

  • 版本：neon-user-20241219-1321.iso
  • 机器：Intel NUC11PAHi5

优点：

  • 如果需要 Ubuntu 生态，推荐！
  • 基于最新的 Ubuntu LTS
  • KDE 团队出品，搭载最适配、最新的 KDE

缺点：

  • KDE 无法精简安装，占用较多（待确定）
